# Marketplace Manager – Milestones

## Milestone 1 – Core Skeleton (Week 1–2)
- Project created with SwiftUI + modular structure.
- Data models stubbed: Product, PlatformListing, Order, PlatformAccount, Earnings.
- Basic Inventory and Orders views reading from mock data.

## Milestone 2 – Auth & Persistence (Week 3–4)
- Keychain-based token storage implemented.
- OAuth redirect + token capture working for eBay sandbox.
- Core Data/SwiftData wired for products and orders.

## Milestone 3 – eBay End-to-End (Week 5–7)
- Create listing in app → listing appears on eBay sandbox.
- Fetch orders from eBay into Orders dashboard.
- Basic earnings calculations using eBay data.

## Milestone 4 – Multi-Platform Support (Week 8–11)
- Mercari listings + orders integrated.
- Facebook Marketplace integration (as permitted by Graph API).
- Single publish flow that can target multiple platforms.

## Milestone 5 – Dashboards & Polish (Week 12–14)
- Earnings dashboard with platform breakdown and date ranges.
- Sync reliability: background timer + manual refresh.
- Error states, toasts/alerts, and basic logging.

## Milestone 6 – Beta Release (Week 15–16)
- Internal beta build for uncle (TestFlight or direct).
- Feedback incorporated into UX and flows.
- Decide next platforms/features for v2 (shipping labels, automation, pricing rules).
